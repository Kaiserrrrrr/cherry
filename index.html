Overview

Cherry.js is a zero-config script that automatically improves website performance, accessibility, and SEO. It acts as a real-time bridge between design intentions and browser requirements.

Key Capabilities

  • Self-Healing Layouts: Fixes layout shifts by calculating missing element dimensions on the fly.
  • Dynamic Semantics: Injects missing ARIA roles and labels into existing HTML.
  • Runtime Optimization: Defers non-critical resources to ensure the fastest time-to-interactive.

Setup

Simply drop the script into your dist folder and reference it in your document head.

<script src="dist/cherry.js"></script>
