Overview

Cherry.js is a zero-dependency optimization engine. It monitors the DOM for common technical debt and applies structural patches during browser idle periods.

Quick Start

Paste the following line into your head:

<script src="dist/cherry.js"></script>

Core Modules

Performance Repair

Identifies images missing width or height attributes and injects them to prevent Cumulative Layout Shift (CLS).

Accessibility

Locates icon-only buttons and unlabelled inputs to provide fallback ARIA descriptors.

SEO & Semantics

Ensures a document contains exactly one h1 and correctly nested landmark roles.
